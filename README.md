# Salesforce Recruitment Management System

## Project Overview
This project automates the recruitment process using Salesforce CRM.

## Features

- Candidate Management
- Job Opening Management
- Application Tracking
- Approval Process
- Record Triggered Flow
- Custom Objects
- Custom Tabs

## Technologies

- Salesforce CRM
- Flow Builder
- Approval Process
- Custom Objects
- Master-Detail Relationship

## Project Phases

- Phase 1 - Object Creation
- Phase 2 - Relationship Setup
- Phase 3 - Approval Process
- Phase 4 - Flow Automation
